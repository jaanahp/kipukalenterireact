Päättöprojekti Careerian tieto- ja viestintätekniikan perustutkinto, datanomi (full stack ohjelmointi)
Azuressa toimiva SQL tietokanta
.NET Core REST.API -rajapinta C# https://github.com/jaanahp/kipukalenteri
Käyttöliittymä React.js https://github.com/jaanahp/kipukalenterireact

Kipukalenteri on tietokantasovellus, jota voi käyttää sekä mobiililaitteella että tietokoneella tai tabletilla.
Sovellukseen on tarkoitus lisätä tietoja omista kiputiloista, esimerkiksi päänsäryistä ja/tai vatsakivuista. 
Sovelluksen tietoja voi lajitella kivun tapahtumisajankohdan tai kivun sijainnin mukaan. Sovellukseen on
toteutettu käyttäjätunnistus, jolloin kukin käyttäjä kirjautuu sisään ja näkee vain omat tietonsa.

Itse suunniteltu ja toteutettu tietokanta, testattu manuaalisesti.
Backendin ja tietokannan välinen yhteys testattu Postman-sovelluksella manuaalisesti sekä automaattinen
integraatiotestaus Alba-kirjastolla.
Frontend testattu manuaalisesti sekä automaattinen yksikkötestaus toteutettu Jest-kirjastolla.
E2E-testaus suoritettu Cypressillä.

Sovelluksen dokumentaatio, testaussuunnitelmat ja testauspöytäkirja sekä näyttöjen arviointipöytäkirjat
liitetty GitHubiin
